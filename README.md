# Space Invaders
## About
Spillet er laget i Pygame (Python). Det har en meny, tre waves med aliens, og seier/game over-skjermer.

## Installation
Open the folder with the files in it. Open a terminal in this folder and write:
```
pip install pygame
```
Then run main.py to play the game

## Tutorial
Start by running the file, then a menu will pop up.

Here you can either click the "Play" button to start the game, or "Quit" to exit.

Then when you've entered the game, use the arrow keys to move your spaceship left and right.

Use spacebar to shoot bullets at the aliens.

Your goal is to eliminate all aliens across 3 waves without letting any reach the bottom.

You die if an alien reaches the bottom of the screen.

## All Keybinds
**Movement:**
- Arrow Left - Move left
- Arrow Right - Move right

**Menu:**
- Left mouse click - click buttons with this key

**Shooting:**
- Spacebar - Shoots bullets (one shot per press)
